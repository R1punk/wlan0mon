### Installation Termux
Quick installation for Termux
```bash
wget https://raw.githubusercontent.com/R1punk/wlan0mon/main/install && bash install
``` 
### Usage:

- ```Root Only```
- ```start``` - Monitor mode
- ```stop``` - Stop Monitor Mode
